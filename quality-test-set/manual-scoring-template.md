## 评分维度

每题按 1-5 分评分：

- 来源支撑：答案和解释是否能被来源片段支持。
- 答案唯一：是否只有一个最合理答案。
- 理解深度：是否考理解、边界、场景、迁移或误区，而不是原文填空。
- 表达清晰：题干、选项、解释是否清楚，没有歧义。
- 干扰项质量：错误选项是否合理但明确错误。
- 复习价值：这道题是否值得用户在未来再次复习。

## 严重问题

出现以下任一情况，标记为严重问题：

- 来源片段无法支持正确答案。
- 存在多个合理正确答案。
- 正确答案明显错误。
- 题目和来源内容无关。
- 解释误导用户。
