今天这段视频主要讲普通人如何开始使用 AI Agent。第一个建议是不要从复杂自动化开始，而是先选择低风险、重复性强、结果容易检查的任务。

比如整理会议纪要、生成资料摘要、对比几个方案的优缺点，这些任务即使出错也比较容易发现。相比之下，直接让 Agent 操作金钱、合同或客户沟通，就需要更强的审核机制。

第二个建议是把任务拆成小步骤。用户先定义目标和边界，再让 Agent 执行其中一个步骤，检查结果后再进入下一步。这样做虽然看起来慢一点，但更容易建立信任。

第三个建议是保留人工确认点。Agent 的价值不是完全取消人的判断，而是把人从重复劳动中解放出来，让人把注意力放在目标、标准和例外情况上。
