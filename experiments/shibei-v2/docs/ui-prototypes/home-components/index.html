连线匹配卡片状态

四种状态复用同一个卡片组件，只改变颜色与交互可用性，不改变尺寸、列宽或排版。

selected 只用于第一次点击的单张卡片；第二次点击后不显示蓝色，直接让两张卡同时进入 correct 或 wrong。

selected：已点选一个选项
correct：匹配正确短反馈
wrong：匹配错误短反馈
locked：正确后锁定不可选
