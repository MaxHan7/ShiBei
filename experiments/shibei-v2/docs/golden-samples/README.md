# V2 Golden Samples

这个目录保存 V2 章节复习流程的人工校准样本。Golden sample 用于后续评估新文章拆分、summary、轻量理解题、场景题、干扰项和误区生成质量。

## 使用方式

评估新文章时，不要求机械复制本样本的题目数量或节点数量。应参考它体现的质量标准：

1. Summary 先点核心命题，再轻量带出展开方向。
2. 知识点按原文阅读顺序切分，背景段不强行出题。
3. 每个知识点形成从开场、轻量理解到场景应用的认知坡度。
4. 有干扰项的题目先生成题级常见误区或混淆点，再生成选项。
5. 匹配/连线题服务于边界辨析或具体化理解，而不是机械对齐名词和解释。
6. 题干考理解，不考原文背诵。
